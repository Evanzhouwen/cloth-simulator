OVERVIEW:
- A Cloth Simulator using Verlet Integration 
- Individual project for Deakin University
- Duration was 1 week

LANGUAGE/API/SOFTWARE:
- C++, DirectX, HLSL
- Visual Studio, Photoshop

FEATURES:
- Cloth can toggle self-collisions
- Pin/unpin any vertex in the cloth
- Change size and statistics of cloth and simulation


=================================================================

 Normal Mode Keys:
	- LMC (on Vertex):    Toggle whether vertex is pinned in space
	- Up/Down Arrows:     Move Camera Forward/Backward
	- Left/Right Arrows:  Move Camera Left/Right
	- Q/W:                Move Camera Up/Down
	- C/V/Z/X/A/S:		  Rotate the Camera

 Handle Mode Keys:
	- LMC (on Vertex):    Toggle whether vertex is pinned in space
	- Up/Down Arrows:     Move Cloth Forward/Backward
	- Left/Right Arrows:  Move Cloth Left/Right
	- Q/W:                Move Cloth Up/Down
	- 1-4:				  Change which row is 'head' of cloth

 Menu Buttons:
  01) Toggle Gravity
  02) Toggle Vertex Visibility
  03) Toggle Object Visibility
  04) Toggle Object Movement
  05) Reset Camera
  06) Reset Cloth
  07) Unpin all pinned vertices
  08) Toggle Cloth Handle Mode
  09) Toggle Cloth self collisions
  10) Toggle Help menu
